## 开源项目

### antd-pro-schema-form（表单引擎）

> **中后台解决方案之表单引擎**
> https://www.npmjs.com/package/antd-pro-schema-form

基于模型驱动思想的 Schema 表单渲染引擎，始于内网沉淀，终于外网开源，反哺于众多内部中后台生产项目。

**核心特性**：
- Schema 驱动：通过 JSON 配置快速生成表单
- 3 层渲染模型：Schema 转换层 → Form.Item 壳子层 → 子组件渲染层
- 设计模式综合运用：策略模式（组件扩展）、代理模式（数据格式化）、观察者模式（通信）
- 嵌入式使用：在已有表单中部分使用表单引擎
- 无限类型扩展：策略模式支持内置子组件无限扩展

### babel-plugin-alias（Babel 插件）

> https://www.npmjs.com/package/babel-plugin-alias

Babel 路径别名插件，简化模块导入路径。
